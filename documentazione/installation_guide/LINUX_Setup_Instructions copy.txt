# Setup Instructions

This guide provides a detailed process for installing all required components for your project on both Windows and Linux. The components include Mosquitto (MQTT broker), MongoDB (database), Python with necessary libraries (paho-mqtt and pymongo), Node.js with npm, and Node-RED with additional packages (node-red-dashboard, node-red-contrib-ui-heatmap, and node-red-node-mongodb).

## Requirements

- Administrative rights on your machine.
- A stable internet connection to download the packages.

## Linux Setup

### 1. Installing Mosquitto (MQTT Broker)

1. Open the terminal and update the package list with:

   sudo apt update

2. Install Mosquitto with:

   sudo apt install mosquitto mosquitto-clients

3. Start and enable the Mosquitto service with:

   sudo systemctl start mosquitto
   sudo systemctl enable mosquitto


### 2. Installing MongoDB

1. Import the public key used by the package management system with:

   wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -

2. Create a list file for MongoDB with:

   echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list

3. Reload the local package database with:

   sudo apt update

4. Install the MongoDB packages with:

   sudo apt install -y mongodb-org

5. Start and enable the MongoDB service with:

   sudo systemctl start mongod
   sudo systemctl enable mongod


### 3. Installing Python and Required Libraries

1. Install Python and pip with:

   sudo apt install python3-pip
   pip3 install paho-mqtt pymongo


### 4. Installing Node.js and npm

1. Install Node.js and npm from the NodeSource repository with:

   curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
   sudo apt install -y nodejs


### 5. Installing Node-RED and Additional Packages

1. Install Node-RED globally with:

   sudo npm install -g --unsafe-perm node-red

2. Install the necessary Node-RED packages with:

   cd ~/.node-red
   npm install node-red-dashboard node-red-contrib-ui-heatmap node-red-node-mongodb

3. Start Node-RED with:

   node-red
