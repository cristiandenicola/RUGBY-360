# Setup Instructions

This guide provides a detailed process for installing all required components for your project on both Windows and Linux. The components include Mosquitto (MQTT broker), MongoDB (database), Python with necessary libraries (paho-mqtt and pymongo), Node.js with npm, and Node-RED with additional packages (node-red-dashboard, node-red-contrib-ui-heatmap, and node-red-node-mongodb).

## Requirements

- Administrative rights on your machine.
- A stable internet connection to download the packages.

## Windows Setup

### 1. Installing Mosquitto (MQTT Broker)

1. Download Mosquitto from the official website: https://mosquitto.org/download/
2. Run the installer and follow the on-screen prompts.
3. Open the command prompt as an administrator and start the Mosquitto service with:

   net start Mosquitto


### 2. Installing MongoDB

1. Download MongoDB from the official website: https://www.mongodb.com/try/download/community
2. Run the installer and follow the on-screen prompts.
3. Open the command prompt as an administrator and start the MongoDB service with:

   net start MongoDB


### 3. Installing Python and Required Libraries

1. Download and install Python from the official website: https://www.python.org/downloads/
2. During installation, make sure to check the box that says "Add Python to PATH".
3. Open the command prompt and install the required libraries with:

   pip install paho-mqtt pymongo


### 4. Installing Node.js and npm

1. Download Node.js from the official website: https://nodejs.org/
2. Run the installer and follow the on-screen prompts.

### 5. Installing Node-RED and Additional Packages

1. Open the command prompt and globally install Node-RED with:

   npm install -g node-red

2. Install the necessary Node-RED packages with:

   npm install node-red-dashboard node-red-contrib-ui-heatmap node-red-node-mongodb

3. Start Node-RED with:

   node-red
